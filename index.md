We build open-source Python tools for UK insurance pricing teams.

The focus is on the gap between what a GBM can do and what a pricing team can actually put into production - factor tables that pass regulatory scrutiny, model outputs that a rating engine can consume, diagnostics that a pricing committee can challenge.

---

## Tools

**[shap-relativities](https://github.com/burningcost/shap-relativities)**
Extract multiplicative rating relativities from CatBoost models using SHAP values. Same format as `exp(beta)` from a GLM - factor tables, confidence intervals, exposure weighting, reconstruction validation.

```bash
uv add "shap-relativities[ml]"
```
